## User Inputs

| *Input*              | *Description*                                                                                  |
| -------------------- | ---------------------------------------------------------------------------------------------- |
| **Name**             | (*Only available to Basic users*) Give the spellbook a name, either typed by you or generated. |
| **Character Level**  | Select the level for the spellbook's owner.                                                    |
| **Class**            | Select the class for the spellbook's owner.                                                    |
| **Schools of Magic** | Select the schools of magic from which to pick spells.                                         |
| **Sources**          | Select the sources from which to pick spells.                                                  |
| **Exclude Legacy**   | Excluding legacy content means only including 2024 content.                                    |
